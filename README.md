# How to Compile and Run cpu.c

# On Linux or Mac (GCC)

Open a terminal in the folder that contains cpu.c and run:

gcc -std=c11 cpu.c -o cpu
./cpu

# On VS Code

1.Install the C/C++ extension
2.Open the folder containing cpu.c
3.Open an integrated terminal
4.Compile: gcc -std=c11 cpu.c -o cpu
